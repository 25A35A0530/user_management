# User Management System

This project is part of my internship task where I implemented dynamic features using PHP and MySQL.

# Features

- User Registration with profile picture upload
- Password hashing using `password_hash()`
- Login & Logout using PHP Sessions
- Role-based authentication (Admin/User)
- CRUD Operations:
  - Add User
  - View Users
  - Edit User
  - Delete User
- Profile Management
- Secure file upload (type & size validation)
- SQL Injection prevention using Prepared Statements

#Technologies Used

- PHP
- MySQL
- HTML & CSS
- XAMPP

# Security Features

- Password encryption
- Server-side validation
- Prepared statements

#How to Run

1. Install XAMPP
2. Start Apache & MySQL
3. Place project in `htdocs`
4. Import database in phpMyAdmin
5. Run: `http://localhost/user_management`
